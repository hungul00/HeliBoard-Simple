# LKSBoard
LKSBoard is a privacy-conscious and customizable open-source keyboard, based on AOSP / OpenBoard.
Does not use internet permission, and thus is 100% offline.

HeliBoard를 fork 하여 영어/한국어만 적용하여 사용하도록 지원하고 있습니다.
